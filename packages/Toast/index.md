# Toast 组件

## 使用

```javascript
<ln-toast v-model:show="showPopup">弹窗内容</ln-toast>
```

## 参数

| 参数       | 描述             | 数据类型 | 默认值 |
| ---------- | ---------------- | -------- | ------ |
| show       | 是否展示         | Boolean  | false  |
| duration   | 展示时间         | Number   | 2000ms |
| position   | 位置             | String   | center |
| canPointBg | 是否可以选中背景 | Boolean  | false  |
